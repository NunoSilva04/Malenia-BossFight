# Graphics_Helper.h

Aggregate of namespaces with complementary functions and debugging tools useful for vulkan.
Uses the third party library [Vulkan SDK](https://vulkan.lunarg.com/sdk/home).

# vk_debug
Responsible for creating, enabling and destroying the vulkan validation layers. These functions only do what is required of them if debug mode is enabled at compile time.

## Member Types
None.

## Member functions
### Instance Functions
#### Function: `bool get_validation_layers(Core::n_vector<const char *> &layers)`
- **Access:** `Public`   
- **Parameters:**  `[in]Core::n_vector<const char *> &layers` - Reference to a vector of strings containing the layer names.  
- **Return:**  `bool`   
- **Description:** Adds `VK_LAYER_KHRONOS_validation` and `VK_LAYER_LUNARG_crash_diagnostic` to the vector and returns true. If it couldn't add at least one of the specified layers, it will return false.

#### Function: `void add_debug_extension(Core::n_vector<const char *> &extensions)`
- **Access:** `Public`   
- **Parameters:**  `[in]Core::n_vector<const char *> &extensions` - Reference to a vector of strings containing the extension names.
- **Return:**  `void`   
- **Description:** Adds `VK_EXT_debug_utils` to the vector and returns.

#### Function: `bool initialize_debug_messenger(const VkInstance vk_instance)`
- **Access:** `Public`   
- **Parameters:**  `[in]const VkInstance vk_instance` - A valid vulkan instance.
- **Return:**  `bool`   
- **Description:** Initializes the debug messenger to print only the warnings and errors generated by the vulkan layers and submits a message to the terminal to ensure it was initialized with success and returns true. Returns false otherwise.

#### Function: `void create_object_name_fn(const VkDevice logical_device)`
- **Access:** `Public`   
- **Parameters:**  `[in]const VkDevice logical_device` - A valid vulkan device.
- **Return:**  `void`   
- **Description:** Initializes the object name utils function to name and keep track of vulkan objects.

#### Function: `void set_object_name(const VkDevice logical_device, VkObjectType type, uint64_t handle, const char *name)`
- **Access:** `Public`   
- **Parameters:**
    - `[in]const VkDevice logical_device` - A valid vulkan device.
    - `[in]VkObjectType type` - The type of the object.
    - `[in]uint64_t handle` - Handle to the object.
    - `[in]const char *` - The name for the object.
- **Return:**  `void`   
- **Description:** Sets the name for the specific object type that was passed.

#### Function: `void destroy_debug_messenger(const VkInstance vk_instance)`
- **Access:** `Public`   
- **Parameters:** `[in]const VkInstance vk_instance` - A valid vulkan instance.
- **Return:**  `void`   
- **Description:** Destroys the debug messenger.


# vk_helper
Library of functions used to print and modify vulkan variables.

## Member Types
### `enum CoreFeatures`
Enumerates all of the core features to be enabled in the device.

| Core Features         | Description         |
| ---------------|---------------|
| Geometry_Shader| Geometry shader device feature  |
| Tesselation_Shader| Tesselation shader device feature  |

### `enum Usage_Flags`
Enumerates all of the wanted usage image flags for the swapchain creation.

| Core Features         | Description         |
| ---------------|---------------|
| COLOR_ATTACHMENT| Color attachment swapchain usage flag  |
| TRANSFER_DESTINATION| Transfer destination swapchain usage flag  |

### `enum Alpha_Flags`
Enumerates all of the wanted alpha flags for the swapchain creation.

| Core Features         | Description         |
| ---------------|---------------|
| OPAQUE| Opaque swapchain alpha flag  |
| PRE_MULTIPLED| Pre multiplied swapchain alpha flag  |
| POST_MULTIPLED| Post multiplied swapchain alpha flag  |
| INHERITED| Inhereted swapchain alpha flag  |
